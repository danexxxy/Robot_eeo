# Система измерения износа электрода ЭЭО для манипулятора

Данный проект реализует систему мониторинга длины электрода электроэрозионного (ЭЭО) привода, установленного как 7-е звено промышленного манипулятора.

## Описание решения
Проект включает в себя:
1. **Динамическую URDF-модель**: Робот с 7 степенями свободы, где длина последнего звена (`Tool`) обновляется без перезагрузки системы.
2. **Модуль измерения**: ROS 2 нода, имитирующая систему компьютерного зрения для определения износа электрода. Реализован алгоритм постепенного износа и автоматической "замены" инструмента.
3. **Визуализацию**: Настроенный запуск в RViz2 с отображением модели робота и виртуального фида с камер.

## Схема размещения камер
Для определения длины используется ортогональная система из двух камер:
* **Camera Front**: Установлена перед рабочей зоной.
* **Camera Side**: Установлена сбоку под углом 90°.
* **Принцип**: Система находит контур электрода, вычисляет его длину в пикселях от патрона до кончика и пересчитывает в метры по калибровочному коэффициенту.

## Установка и сборка
Требуется установленный ROS 2 Humble.

```bash
# Создание рабочего пространства и клонирование
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws/src
# (склонируйте сюда папку eeo_robot)

# Сборка пакета
cd ~/ros2_ws
colcon build --packages-select eeo_robot
source install/setup.bash
```

## Запуск
Запуск всей системы (измерение + робот + RViz):
```bash
ros2 launch eeo_robot display_rviz.launch.py
```

## Тестирование
Проект включает набор Unit-тестов для проверки математики расчета и логики замены электрода:
```bash
colcon test --packages-select eeo_robot
colcon test-result --all --verbose
```
